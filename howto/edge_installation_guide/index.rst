Edge安装指引
==========================

.. rubric:: 前提
---------------------------

- 你需要预先设置好从Edge安装环境到EnOS的VPN连接。

- OU必须将你的服务器准备好，以安装Edge。EnOS Edge推荐的硬件型号，参见附录。

- 你需要确保硬件服务器的操作系统是Linux CentOS7.4。
