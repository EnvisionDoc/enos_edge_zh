Web配置中心
====================================

在EnOS Edge连接IoT设备之前，需要在EnOS上为Edge本身以及这些设备创建模型、产品、和设备。 更多信息，参见`设备接入相关文档 <https://www.envisioniot.com/docs/device-connection/zh_CN/latest/device_management_overview.html>`_。

完成以上步骤之后，你需要在 **Edge网关** 为注册该Edge并添加连接、创建模板和规约，方可使用Edge收集设备数据以及其他功能。
